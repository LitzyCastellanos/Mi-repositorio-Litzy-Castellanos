Descripción del proyecto

Este proyecto es un sistema de clasificación de pacientes en triaje médico basado en una Red Neuronal Artificial (RNA) desarrollada en Python con TensorFlow.

El modelo predice el nivel de prioridad de atención médica (ESI 1 a ESI 5) a partir de datos clínicos como signos vitales, laboratorios y motivo de consulta. La aplicación incluye una interfaz gráfica (GUI) creada con Tkinter que permite ingresar datos de un paciente y obtener una predicción en tiempo real.

¿Qué es el sistema ESI?

El Emergency Severity Index (ESI) clasifica a los pacientes según la urgencia:

Nivel Descripción Color
ESI 1 Crítico (riesgo vital inmediato) ● Rojo
ESI 2 Emergencia ● Naranja
ESI 3 Urgente ● Amarillo
ESI 4 Menos urgente ● Verde
ESI 5 No urgente ● Azul

¿Cómo funciona?

  1. Se carga un conjunto de datos clínicos de pacientes.
  2. Se limpian y procesan los datos.
  3. Se convierten variables categóricas en numéricas.
  4. Se entrena una red neuronal multicapa.
  5. El modelo aprende a clasificar el nivel ESI.
  6. La interfaz permite ingresar datos de un paciente.
  7. El sistema predice el nivel de urgencia en tiempo real.

Características del modelo

  • Red neuronal con capas densas (Dense layers)
  • Dropout para evitar sobreajuste
  • Batch Normalization para estabilizar el entrenamiento
  • Early Stopping para detener el entrenamiento cuando converge
  • Interfaz gráfica (Tkinter) para predicciones en tiempo real
  • Indicador visual por color según nivel ESI

Interfaz del sistema

Al ejecutar el programa se abre una ventana donde puedes ingresar:

  • Edad
  • Sexo
  • Presión arterial
  • Frecuencia cardíaca
  • Temperatura
  • Laboratorios
  • Motivo de consulta

Luego presionas "Predecir nivel ESI" y el sistema mostrará:

  • Nivel ESI (1–5)
  • Probabilidades por clase
  • Indicador visual por color

Tecnologías utilizadas

Python 3.11 TensorFlow / Keras Pandas NumPy Scikit-learn Tkinter CSV Dataset

Guía de instalación paso a paso

Requisitos previos

  • Python 3.11 específicamente (requerido)
  • pip (gestor de paquetes de Python)
  • Git instalado
  • Verificar versión: python --version
  1. Clonar el repositorio
    git clone https://github.com/LitzyCastellanos/Sistema-de-Triaje-RNA.git
    cd Sistema-de-Triaje-RNA
  2. Instalar dependencias
    pip install pandas numpy scikit-learn tensorflow
  3. Ejecutar el sistema
    python triaje_RNA.py
  4. Usar la interfaz
    Ingresar datos del paciente → Clic en "Predecir nivel ESI" → Ver resultado
