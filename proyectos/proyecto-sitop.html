Descripción general del sistema

Desarrollo completo del sitio web institucional para SITOP, una empresa hondureña de ingeniería civil con amplia trayectoria en servicios de construcción, topografía de precisión y supervisión de obras. El sitio funciona como carta de presentación digital y plataforma de contacto para potenciales clientes.

Construido completamente con HTML5 semántico, CSS3 moderno y JavaScript vanilla, sin dependencias externas. Diseño totalmente responsive, optimizado para SEO y con rendimiento superior al 95% en Lighthouse.

Descripción de páginas

Página Contenido
Inicio Hero slider con imágenes de obras, servicios destacados, proyectos recientes y sección de contacto.
Quiénes somos Historia de la empresa, misión, visión y valores institucionales con timeline.
Servicios Catálogo completo: construcción, topografía, supervisión de obras, planificación y consultoría.

Contacto Formulario de contacto funcional, datos de la empresa, mapa de ubicación y redes sociales.

Funcionalidades destacadas

  • Hero Slider: Carrusel automático con imágenes de obras realizadas y transiciones suaves.
  • Galería filtrable: Sistema de filtros por categoría con animaciones CSS.
  • Modal de proyectos: Ventana emergente con detalles completos de cada obra al hacer clic.
  • Formulario de contacto: Validación en tiempo real con JavaScript.
  • Navegación sticky: Menú que se oculta al hacer scroll down y reaparece al hacer scroll up.
  • Scroll to top: Botón flotante para volver al inicio de la página.
  • Animaciones: Reveal on scroll, hover effects en tarjetas y transiciones suaves.
  • Mobile-first: Diseño adaptable con menú hamburguesa y galería responsive.

Tecnologías utilizadas

HTML5 Semántico CSS3 (Grid + Flexbox) JavaScript Vanilla Responsive Design CSS Animations SEO On-Page Netlify Deploy GrupoHost Domain

Guía de instalación paso a paso

Requisitos previos

  • Cualquier navegador web moderno (Chrome, Firefox, Edge, Safari)
  • Editor de código (VS Code recomendado)
  • Git (para clonar el repositorio)
  • Cuenta en Netlify (para deploy)
  1. Clonar el repositorio
    git clone https://github.com/LitzyCastellanos/sitop-web.git
    cd sitop-web
  2. Abrir en el navegador
    Doble clic en index.html o usar Live Server en VS Code.
  3. Personalizar contenido
    Editar textos en los archivos .html y cambiar imágenes en la carpeta /img.
  4. Desplegar en Netlify
    Arrastrar la carpeta del proyecto a Netlify o conectar el repositorio de GitHub para deploy automático.
  5. Configurar dominio personalizado
    En Netlify → Domain Settings → Add custom domain → sitophn.com (adquirido en GrupoHost).

Despliegue

  • Hosting: Netlify (gratuito con SSL incluido)
  • Dominio: sitophn.com adquirido en GrupoHost
  • SSL: Certificado automático de Netlify
  • CI/CD: Deploy automático al hacer push a la rama main

Notas importantes

  • El sitio es 100% responsive gracias al uso de Flexbox, Grid y media queries.
  • Las animaciones están optimizadas para no afectar el rendimiento.
  • El formulario de contacto incluye validación frontend con JavaScript.
  • El slider del hero es automático pero también permite navegación manual.
  • La galería de proyectos se puede filtrar por categoría (construcción, topografía, supervisión).
  • Se recomienda optimizar las imágenes antes de subirlas para mejorar el rendimiento.
