## Descripcion

Portafolio web personal desarrollado desde cero con HTML5, CSS3 y JavaScript puro, sin frameworks ni librerias externas. Presenta mis proyectos, habilidades y experiencia como Ingeniera en Sistemas y desarrolladora frontend.

---

## Tecnologias

| Tecnologia | Uso |
|---|---|
| HTML5 | Estructura semantica |
| CSS3 | Grid, Flexbox, animaciones, variables CSS |
| JavaScript | Interactividad sin dependencias |
| Vercel | Hosting y despliegue continuo |

